Definitions

A Browser
is a program used to search for and display web pages.
Cookies
are information files created when you visit a site and contain a unique, anonymous number. They are stored in the cookie directory of your hard drive and do not expire at the end of your session.
An Internet Protocol address
is a technical standard which allows data to be transmitted between two devices.
A Webserver
delivers web pages to a computer, in response to a request from a Browser.
